# OpenClaw Metaverse - Godot 測試策略

**負責：** Nora（QA Tester）
**更新：** 2026-04-06
**專案：** OpenClaw Metaverse（Godot 4.x）

---

## 一、測試目標

確保 Godot 3D 元宇宙遊戲的核心功能穩定運作，特別是 Agent 互動系統。

---

## 二、測試環境

| 項目 | 環境 |
| --- | --- |
| Engine | Godot 4.6 |
| 平台 | macOS |
| GUI | 需要（無法 headless 測試）|
| 輸入 | WASD + 滑鼠 + 鍵盤 |

---

## 三、測試範圍

### P0 測試（核心功能）

#### 1. 玩家移動
```
| 測項 | 方法 | 預期結果 |
| --- | --- | --- |
| WASD 移動 | 按 W/A/S/D | 玩家在 3D 世界移動 |
| 攝影機視角 | 滑鼠拖曳 | 視角旋轉 |
| 縮放 | 滑鼠滾輪 | 縮放場景 |
```

#### 2. Agent 互動系統
```
| 測項 | 方法 | 預期結果 |
| --- | --- | --- |
| 接近 Agent | 走到 Agent 附近 | 顯示提示「按 E 互動」 |
| E 鍵開啟對話 | 按 E 鍵 | 開啟對話 UI |
| 對話輸入 | 輸入指令 | Agent 回應 |
| ESC 關閉 | 按 ESC | 關閉對話 UI |
```

#### 3. Boss UI 面板
```
| 測項 | 方法 | 預期結果 |
| --- | --- | --- |
| 面板顯示 | 點擊 Agent | 右上角顯示面板 |
| 指令執行 | 點擊指令按鈕 | Agent 執行命令 |
| 關閉面板 | 點擊關閉/X | 面板消失 |
```

#### 4. Agent 狀態機
```
| 測項 | 方法 | 預期結果 |
| --- | --- | --- |
| IDLE 狀態 | Agent 待命 | 漂浮動畫 |
| WALKING 狀態 | Agent 移動 | 行走動畫 |
| TALKING 狀態 | 對話中 | 對話泡泡顯示 |
| WORKING 狀態 | 執行任務 | 工作動畫 |
```

### P1 測試（重要功能）

#### 5. 9 房間系統
```
| 測項 | 方法 | 預期結果 |
| --- | --- | --- |
| 房間入口 | 走到房間入口 | 顯示房間名稱 |
| 房間切換 | 進入房間 | 場景切換 |
```

#### 6. 對話泡泡
```
| 測項 | 方法 | 預期結果 |
| --- | --- | --- |
| 泡泡顯示 | Agent 說話 | 浮空泡泡出現 |
| 泡泡消失 | 對話結束 | 泡泡淡出 |
```
